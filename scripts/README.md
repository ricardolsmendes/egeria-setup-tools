# egeria-setup-tools

## Scripts

1. Turn all scripts executable:

   ```sh
   cd scripts
   chmod +x *sh
   ```

1. Pull the latest Docker image:

   ```sh
   ./pull-omag-server-platform-docker-image.sh
   ```

1. Run the OMAG Server Platform using Docker:

   ```sh
   ./run-omag-server-platform-docker.sh <HOST-LIBS-DIRECTORY>
   ```
